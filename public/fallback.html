This is a fallback page that loads when the main React application has issues.

If you're seeing this page, it means the React app is having trouble initializing.

Troubleshooting Steps

  1. Open browser developer tools (F12) and check the console for errors
  2. Verify that your Firebase configuration is correct
  3. Check that all required environment variables are set
  4. Try clearing your browser cache and reloading
